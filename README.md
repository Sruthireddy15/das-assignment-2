# das-assignment-2
This repository contains five C programs that illustrate practical applications of stacks and queues using array‑based implementations. It includes a program that reverses a string by pushing each character onto a stack and then popping them back in reverse order, and another that checks whether parentheses in a mathematical expression are balanced by matching every opening bracket with a corresponding closing bracket. The repository also features a program that finds the next greater element for each item in an array using an efficient stack‑based approach, a simulation of a printer queue where documents are added, printed in order, and the pending list can be viewed, and finally a menu‑driven circular queue implementation that supports enqueue, dequeue, peek, and display operations while reusing array space through circular index wrapping.
